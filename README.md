# Serenatto - Sistema de Cardápio Digital

## Descrição do Projeto

O Serenatto é um sistema de cardápio digital desenvolvido para um café e restaurante. Ele permite que os clientes visualizem as opções de café da manhã e almoço de forma fácil e intuitiva, além de fornecer uma interface de administração para gerenciar os produtos.

## Funcionalidades

- Exibição de produtos separados por categorias (Café e Almoço)
- Painel de administração para gerenciar produtos
- Conexão com banco de dados MySQL para armazenamento de informações

## Tecnologias Utilizadas

- PHP
- MySQL
- HTML
- CSS

## Estrutura do Projeto

- `index.php`: Página principal que exibe o cardápio digital
- `admin.php`: Interface de administração para gerenciar produtos
- `src/conexao.php`: Arquivo de conexão com o banco de dados
- `css/`: Diretório contendo os arquivos de estilo
- `img/`: Diretório para armazenamento de imagens
